1. Procesamiento y Almacenamiento

CPU (Procesador)

Creador: Marcian Hoff, Federico Faggin e Intel (1971)

El "cerebro" de la computadora. Se encarga de interpretar, ejecutar las instrucciones de los programas y coordinar el flujo de datos mediante operaciones aritméticas y lógicas esenciales.

Memoria RAM

Inventor: Robert Dennard (1966)

Memoria de acceso aleatorio y volátil. Almacena temporalmente los datos de los programas que se están ejecutando en tiempo real para que el procesador acceda a ellos de forma ultra rápida.

Placa Base / Tarjeta Madre

Creador: IBM (1981)

La placa de circuito impreso principal de la computadora. Actúa como el sistema nervioso central, interconectando mecánicamente y comunicando electrónicamente la CPU, las memorias, los discos y demás periféricos.

Memoria ROM (Read Only Memory)

Desarrollo: Intel / Toshiba (Evolución de semiconductores)

Memoria de solo lectura y no volátil. Contiene instrucciones y datos de fábrica esenciales e imborrables que el sistema necesita para iniciar el equipo y realizar el diagnóstico primario (firmware/BIOS).

Disco Duro (HDD) / SSD

Creadores: Reynold B. Johnson (HDD, 1956) / Toshiba (Flash, 1980s)

Dispositivos de almacenamiento secundario permanentes. El HDD guarda los datos magnéticamente en discos giratorios, mientras que el SSD utiliza arreglos electrónicos de memoria flash.

2. Fundamentos de Lógica y Software

Algoritmos

Precursor: Al-Juarismi (Siglo IX) / Ada Lovelace (Primer algoritmo, 1843)

Conjunto finito, ordenado y secuencial de instrucciones o pasos lógicos bien definidos que permiten resolver un problema, procesar datos o llevar a cabo una tarea o cálculo específico.

Lógica Algorítmica

Teoría: George Boole (Álgebra Booleana, 1854) / Alan Turing

Metodología y estructura de pensamiento abstracto utilizada para diseñar secuencias lógicas efectivas. Se basa en reglas de inferencia, operadores booleanos (AND, OR, NOT) y estructuras de control para guiar el flujo de un programa de forma matemática.

3. Clasificación de Sistemas

Tipos de Computadoras

Clasificación: Evolución de la Arquitectura de Computadores

Segmentación jerárquica de los equipos según su potencia, tamaño y arquitectura: Supercomputadoras (cálculo científico masivo), Mainframes (procesamiento de datos corporativos masivos), Computadoras Personales / PC (escritorio y laptops de uso general) y Sistemas Embebidos (microcontroladores dedicados en electrodomésticos o autos).

4. Periféricos de Entrada y Salida

Monitor

Inventor: Karl Ferdinand Braun (1897)

Periférico de salida que traduce las señales de video provenientes de la tarjeta gráfica en imágenes legibles para el usuario, permitiendo la interacción visual con el sistema.

Teclado y Ratón

Inventores: C. Sholes (Teclado) / Douglas Engelbart (Ratón, 1964)

Periféricos de entrada primarios. El teclado procesa comandos de texto estructurado y atajos, mientras que el ratón gestiona el movimiento espacial en un entorno gráfico tridimensional o bidimensional.
